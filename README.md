# triipholic-tours-website
A modern premium travel agency website featuring smooth animations, dark red luxury UI, and enquiry form data storage in Google Sheets via Google Apps Script.

This project is a modern tours and travel website designed with a premium dark-themed UI and smooth scroll animations. The platform allows users to explore travel packages and submit enquiries, which are stored directly in Google Sheets using Google Apps Script, eliminating the need for a traditional backend server.


Tech Stack:
- React + Vite
- Tailwind CSS
- Framer Motion
- Google Apps Script
- Google Sheets (Data Storage)
